## 一、核心逻辑：体力耗尽 → 战绩总结 → 付费转化

**不在用户最焦虑的时候弹付费墙。** 体力耗尽的那一刻，用户想的是"我还想再练一题"——这时候弹付费就像在堵车时收过路费。

正确做法：体力耗尽 → 展示"今日战绩总结"（让用户感到成就感）→ 顺势引导付费。

```
体力用完
    │
    ▼
┌─────────────────────┐
│  🎯 今日战绩         │
│  ✅ 正确 7/10        │
│  🔥 连斩 Combo ×3    │
│  👻 1题进入错题池    │
│                     │
│  想继续刷题？        │
│  [ $4.99/月 无限刷 ]  │  ← 主力转化
│  [ 继续免费错题 ]     │  ← 不付费也能留
└─────────────────────┘
```

## 三、每档的设计理由

### Free — 不是阉割版，是健身房免费体验

- 10体力/天 = 10道何切题。对于休闲用户够用，对于硬核用户不够——他们一天刷50题是常态。
- **错题永远免费重练**——这是留存引擎。免费用户不会因为体力耗尽而离开，他们永远可以练错题。但错题只有几道，想练新题？付费。

### Monthly $4.99 — 主力收入

- 对标 Chess.com Gold ($5/月)，价格持平。
- 零竞争——没有第二个日麻训练App，用户没地方比价。
- $4.99/月 × 500付费用户 = $2,495 MRR。小市场做深度不做广度。

### Annual $29.99 — 性价比锚

- 月卡×5 = 半价折扣。"打5个月就回本"。
- 年卡用户流失率低，LTV 最稳。

### Lifetime $49.99 — 铁粉收割

- 对硬核玩家：打一年月卡 = $60 > $49.99 → 永久更划算。
- 对轻度用户：不会买，不影响。对铁粉：买了之后零流失。

## 四、体力消耗设计

| 行为 | 消耗 | 为什么 |
|------|:--:|------|
| 何切实战（1题） | 1体力 | 核心消耗 |
| 闪卡复习（1组） | 1体力 | 学习消耗 |
| **错题重练** | **0体力** | 留存引擎——永远免费 |
| **每日挑战（3题）** | **0体力** | 每日回访钩子 |

免费用户每天至少能完整参与：3题每日挑战 + 10题何切 + 不限次错题重练。活跃度够高，但一定会遇到"想练新题但没体力了"的时刻——那就是付费点。

## 六、收入预估

| 阶段 | DAU | 付费率 | 付费用户 | MRR |
|------|:--:|:--:|:--:|------|
| 冷启动（1-2月） | 200 | 4% | 8 | ~$50 |
| 增长期（3-6月） | 1,000 | 5% | 50 | ~$250 |
| 成熟期（6-12月） | 5,000 | 6% | 300 | ~$1,500 |

总市场规模就5-10万人，天花板低但不代表不能做。月入$1,500-$2,000对于个人开发者是可持续的副业收入。
