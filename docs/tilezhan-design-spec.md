## 一、设计哲学

### 1.1 四大设计原则

| 原则 | 含义 | 设计体现 |
|---|---|---|
| **瞬时识别** | 0.5秒内认出牌面 | 大图卡片 + 高对比度 + 色彩编码 |
| **一刀斩断** | 正反馈极致爽快 | 碎裂粒子 + 武士刀音效 + 金色爆发 |
| **幽灵追猎** | 错题永不放过 | 红色震动 → 强制翻转 → 错题池幽灵标记 |
| **渐进淡出** | 辅助线随等级消失 | Mnemonic标注从100% opacity → 0% over Lv.1→Lv.10 |

### 1.2 情感曲线设计

```
用户情绪
  ↑
  │    🔴 答错受挫
  │   / \              🟢 一刀斩爽感
  │  /   \    _______/¯¯¯¯¯¯¯¯¯
  │ /     \__/                  ← 平稳刷题区
  │/ 新鲜感
  └──────────────────────────────→ 时间
     闪卡    何切    错题     结算
```

**关键干预点**：
- 连续答错 3 次 → 自动弹出"助记插画提示"（硬干预）
- 连续答对 5 次 → 触发"连斩 Combo"特效（正向强化）
- 体力值耗尽 → 不立即弹付费墙，先展示"今日战绩总结"（软着陆）

### 2.2 字体层级

| 层级 | 字号 | 字重 | 行高 | 用途 | 字体 |
|---|---|---|---|---|---|
| **H1** | 32px | Bold (700) | 40px | 页面大标题 (如"今日任务") | Poppins / Noto Sans |
| **H2** | 24px | SemiBold (600) | 32px | 模块标题 | Poppins |
| **H3** | 20px | SemiBold (600) | 28px | 卡片标题 | Poppins |
| **Body** | 16px | Regular (400) | 24px | 正文/说明文字 | Inter / Roboto |
| **Body-Small** | 14px | Regular (400) | 20px | 辅助说明 | Inter |
| **Caption** | 12px | Medium (500) | 16px | 牌面微标注 / 标签 | Inter |
| **Number** | 28px | Bold (700) | 36px | 分数 / 倒计时 / 番数 | JetBrains Mono |
| **汉字牌面** | 48px | Bold (700) | - | 闪卡主字 | Noto Serif SC |

**字体导入策略**：
- 英文字体打包进 App（Poppins + Inter + JetBrains Mono ~500KB）
- 中文字体使用系统默认（iOS: PingFang SC, Android: Noto Sans SC）

### 2.3 间距系统 (8px Grid)

| Token | 值 | 用途 |
|---|---|---|
| `space-xs` | 4px | 图标与文字紧贴 |
| `space-sm` | 8px | 组件内间距 |
| `space-md` | 16px | 卡片内边距 / 列表间距 |
| `space-lg` | 24px | 模块间距 |
| `space-xl` | 32px | 页面区块间距 |
| `space-2xl` | 48px | 大模块分隔 |
| `space-3xl` | 64px | 页面级间距 |

### 2.4 圆角与阴影

```
圆角 (Border Radius):
  sm:  8px   — 按钮 / 标签 / 小卡片
  md:  12px  — 闪卡 / 选项按钮
  lg:  16px  — 主卡片 / 弹窗
  xl:  24px  — 大面板 / 底部Sheet
  full: 999px — 药丸按钮 / 徽章

阴影 (Box Shadow) — 赛博霓虹发光体系:
  glow-gold:   0 0 20px rgba(255,215,0,0.4)   — 胡牌特效
  glow-red:    0 0 16px rgba(255,59,48,0.3)   — 错误震动伴随
  glow-green:  0 0 12px rgba(10,47,29,0.8)    — 卡片悬浮
  card-shadow: 0 4px 24px rgba(0,0,0,0.5)     — 卡片投影
```

│  │      🏖️                  │   │
│  └──────────────────────────┘   │
│  ┌──────────────────────────┐   │
│  │  C.  60,000              │   │
│  │      🚀                  │   │
│  └──────────────────────────┘   │
│  ┌──────────────────────────┐   │
│  │  D.  70,000              │   │
│  │      🪝                  │   │
│  └──────────────────────────┘   │
│                                  │
│     [长按查看助记插画]            │  ← 半透明提示文字
│                                  │
└──────────────────────────────────┘

四选一点击后的反馈动效:

  正确 (选B = 五万 = 50,000):
    0ms:   选项B边框变霓虹金
    150ms: 牌面"五萬"下方浮现沙滩椅插画 (Mnemonic Layer)
    300ms: 插画播放微动效 (躺椅摇晃, 可乐冒泡)
    500ms: 绿色脉冲扩散, 弹出 "✓ 50,000!"
    700ms: 自动切换到下一张牌

  错误 (选了C = 六万 = 60,000):
    0ms:   选项C边框变琉璃红
    100ms: 牌面剧烈震动 (Haptic: 3次短震)
    200ms: 弹出大红色 ✗, 覆盖错误选项
    400ms: 正确答案B自动高亮为霓虹金
    600ms: 牌面强制翻转 → 展示"五 = 沙滩椅"助记插画 + Slogan
    1000ms: 翻转回正面, 用户必须手动点击"Got it"确认
    1200ms: 自动加入错题本, 切换到下一张

左右滑动模式 (Tinder-style):
  - 右划 = "认识" → 绿色✓浮层 → 记入正确池
  - 左划 = "不认识" → 红色✗浮层 → 触发翻转讲解
  - 上划 = "熟练掌握" → 金色⚡浮层 → 永久移出该牌的复习池
```

## 五、核心交互模式

### 5.1 "一刀斩"交互规范 (The Slash)

**触发条件**：何切模块中用户选择了最优解（Perfect判定）

**动效序列**：
```
Time  |  视觉                  |  音频              |  触觉
──────┼───────────────────────┼───────────────────┼─────────
0ms   | 选中牌高亮霓虹金       |  "唰—" (蓄力音)    |  无
150ms | 刀光斜切 (45° 左上→右下)|  "CHING!" (斩击音) |  单次强震
300ms | 牌面裂成两半，碎片飞散  |  碎片飞溅音         |  连续微震
500ms | 金色粒子从裂缝爆发      |  "HU—!" (胡牌音)   |  无
800ms | "PERFECT" 大字弹入     |  胜利短号角         |  无
1200ms| 进张数据展示           |  无                |  无
```

**刀光实现 (Flame)**：
- 使用 `SpriteAnimationComponent` 播放 4 帧刀光序列图（16ms/帧）
- 粒子系统配置：金色 `#FFD700`，80个粒子，初始速度 200-400px/s，扩散角 90°-180°，生命期 600ms，带重力衰减

### 5.2 "错题翻转"交互规范 (The Flip)

**触发条件**：闪卡模块中用户选择错误答案

**动效序列**：
```
Time  |  视觉                  |  音频         |  UI状态
──────┼───────────────────────┼──────────────┼─────────
0ms   | 错误选项变红           |  低音"嗡—"   |  选项锁定
100ms | 牌面剧烈震动 (±8px, 3次)|  震动音效     |  其余选项变灰
300ms | 正确答案霓虹金高亮      |  "叮!"       |  用户不可操作
600ms | 牌面3D翻转 (rotateY)   |  翻页音      |  翻转中
900ms | 背面: 助记插画+Slogan   |  无          |  展示中
      | 强制停留 ≥ 2秒        |              |
3000ms| 翻转回正面             |  翻页音      |  [Got it] 按钮可用
```

### 5.3 卡片滑动 (Swipe-to-Learn)

**手势区域**：屏幕中央 60% 区域

**阈值**：
- 水平滑动 ≥ 60px 或速度 ≥ 300px/s → 触发射出动画
- 水平滑动 < 60px → 回弹原位

**视觉反馈**：
- 右划：绿色渐变覆盖层（opacity 随位移从 0→0.5）
- 左划：红色渐变覆盖层
- 上划：金色渐变覆盖层 + ⚡图标放大

### 5.4 体力值系统交互

```
初始: ❤️❤️❤️ (3/3)
