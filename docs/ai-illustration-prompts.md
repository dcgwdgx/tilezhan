## 使用说明

1. 复制上面任意一段 prompt
2. 粘贴到 Midjourney / DALL-E 3 / Stable Diffusion / Ideogram
3. 建议添加通用后缀参数（根据工具不同）：

```
Midjourney: --style flat --ar 2:3 --no text,letters,watermark
DALL-E 3:    flat vector illustration, no text, 2:3 aspect ratio
SD/Ideogram: flat vector, dark background, bold outlines, cartoon style, no text
```

4. 生成后裁剪为 512×768，底部 80px 留白（用于代码里叠加文字）
5. 输出格式选 SVG 优先，PNG 作为 fallback
